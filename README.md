## Setup Instructions

1. **Ensure logo is placed** in the `assets/` folder as `logo.png`
2. **Open index.html** in a web browser to view the website
3. **Test all pages** by navigating through the menu
4. **Test contact form** functionality (simulated submission)

## Customization

### Logo
- Replace `assets/logo.png` with your company logo
- Recommended size: 200x40 pixels for optimal display

### Colors
The main brand color is `#e74c3c` (red). This can be changed in the CSS file:
- Primary buttons: `.btn-primary`
- Accent colors: Various elements throughout the site

### Content
- Update company information in each HTML file
- Modify services and products as needed
- Adjust contact information in the footer and contact page
